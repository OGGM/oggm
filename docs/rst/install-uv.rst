OGGM now also supports installation with `uv`_, an ultra-fast package manager
which allows you to quickly switch between different installations of OGGM,
and set up reproducible environments for running simulations.

We would recommend uv for more advanced users / developers whow know what they
are doing with non-python OGGM dependencies such as GDAL.

To install `uv`:

.. code-block:: console

    wget -qO- https://astral.sh/uv/install.sh | sh

.. _uv: https://docs.astral.sh/uv/getting-started/features/
