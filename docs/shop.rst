OGGM Shop
=========

.. figure:: _static/logos/logo_shop.png
    :width: 100%

OGGM requires several datasets to run. **All required data are open-access
and can be downloaded automatically**. Because these datasets need to be
extracted and pre-processed for each glacier, OGGM provides a service we
call the "shop", which allows users to download "ready-to-go" :doc:`shop-preprodirs` or
add :doc:`shop-datasets` to their glacier directories "on demand".

The available products range from the essential inputs required for the
standard OGGM workflow such as topography to additional datasets that might
be useful forspecific applications, such as glacier velocity fields.

.. important::

    Don't forget to set up or check your system (:ref:`system-settings`)
    before downloading new data! This only needs to be done once per
    computer.
